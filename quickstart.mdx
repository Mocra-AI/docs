## Getting started

### Step 1: Choose and set up your environment

<AccordionGroup>
  <Accordion icon="terminal" title="HTTP API">
    Mocra has a powerful RESTful HTTP API accessible at `https://api.mocra.io/`.
    You can use it to interact with Mocra if you prefer an HTTP interface or if
    your favorite programming language is not listed below.

    Using the HTTP API does not require installing anything as long as your
    language or environment can perform HTTP operations.

  </Accordion>
  <Accordion icon="node" title="NodeJS SDK">
    If you prefer a SDK, currently we provide a driver for NodeJS. 1. Install
    the Mintlify CLI: `npm i -g mint` 2. Navigate to your docs directory and
    run: `mint dev` 3. Open `http://localhost:3000` to see your docs live!

    To use the NodeJS SDK, you will have to install it. Run the following command:
    ```bash
    npm install @mocra/observe
    ```

    This installation includes TypeScript typings in case you want to use them.

  </Accordion>
  <Accordion icon="python" title="Python SDK">
    We currently do not have a native Python SDK, but it will be coming soon.
    Stay updated!
    <Tip>
      Suscribe to our newsletter by emailing `contact@mocra.io` to receive
      updates regarding development.
    </Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Get a Mocra API key

An API key will be required to interact with Mocra.

Currently, Mocra is in beta trial, so a limited number of API keys is being issued to selected customers.
If you are interested in using Mocra, email `contact@mocra.io` to join our waitlist.

### Step 3: Write your code

<AccordionGroup>
  <Accordion icon="terminal" title="HTTP API">
    You can use the HTTP API from any environment that supports HTTP requests. Here there is an example using cURL:
    ```bash
    curl \
      -X POST \
      -H "Authorization: Bearer <YOUR API KEY>" \
      -H "Content-Type: application/json" \
      -H "Accept: application/json" \
      -d "{\
          \"videoUrl\": \"<YOUR VIDEO URL>\",\
          \"customCriteria\": [],\
          \"removeCriteria\": []\
      }"\
      https://api.mocra.io/observe
    ```
  </Accordion>
  <Accordion icon="node" title="NodeJS SDK">
    Import required functions from `@mocra/observe` and use as in the them that follows:
    ```typescript
    import { VideoObservabilityApi } from "@mocra/observe";

    new VideoObservabilityApi("<YOUR API KEY>") // Replace with your actual API key
      .scoreVideo(
        "http://example.org/video.mp4", // Replace with your actual video to analyze
        [
          {
            criterionName: "Blur",
            criterionDescription:
              "The video should not be blurry",
          },
        ],
        ["UNNATURAL PHYSICS"],
      )
      .then((result) => console.log(result));
    ```

  </Accordion>
  <Accordion icon="python" title="Python SDK">
    We currently do not have a native Python SDK, but it will be coming soon.
    Stay updated!
    <Tip>
      Suscribe to our newsletter by emailing `mocrateam@gmail.com` to receive
      updates regarding development.
    </Tip>
  </Accordion>
</AccordionGroup>

<Tip>
  Mocra uses Mintlify to remain optimized for LLMs. Feel free to use your
  favorite LLM to write code that interacts with Mocra!
</Tip>
